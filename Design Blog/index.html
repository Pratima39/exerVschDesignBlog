What is an anchor tag?

12/01/2020

Here is a great article all about anchor tags.

"An anchor tag is an HTML tag. It is used to define the beginning and end of a hypertext link. Search engines use the tag to determine the subject matter of the destination URL. Users click on the anchor text to reach the link target."

Anchor tags were used in this webpage. Clicking the home button takes you to the top of the page. Click About Me and Contact in the Header and see where they take you. Don't forget to also look at the source code of this example and see if you can figure out how it works. The article linked above explains it if you need more direction.

How to get that fancy wrapped-text-around-an-Image look

Float is a great property. Here is the reference code I used to wrap the text around the images in each post. That is the beauty of the internet: there are lot's of resources like this one available at your fingertips. You may also notice in my code that I made a floated-left and a floated-right id. All of the images used came from pixabay.com. You could check it out here. The images are free to use for personal or commercial uses and do not need to be cited. But it is nice to do so when you can. There are lots of other free stock image sites as well that a quick google search shows.

After I find some images that give me a sense of direction, I like to pick fonts. Google Fonts is a great resource. To grab a font from Google Fonts:

  • Click on one you like
  • Click the "Select this style" button next to the font style
  • You can add as many different fonts/styles of the same font at this point as you want
  • Click Embed in the upper right hand corner
  • Copy the link for your html, or import the font to the css at the top directly
  • copy the css provided to use the font in your css
